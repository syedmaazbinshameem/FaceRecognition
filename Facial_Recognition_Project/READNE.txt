How to use this Program:


1. Download the Facial_Recognition_Project on your Local PC
2. Make sure you have Python installed with the following libraries:
    numpy
    opencv
    Pillow

3. To install the required libraries, please paste the following commands in your Command Prompt/Terminal
    pip install numpy
    pip install Pillow
    pip install opencv-contrib-python

4. Open your command prompt and type the following command: (Make sure you are in the Facial_Recognition_Project folder)
    python faces.py
